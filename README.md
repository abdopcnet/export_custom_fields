# Export Custom Fields

![Version](https://img.shields.io/badge/version-19.1.2026-blue)

Export Frappe customizations to JSON files for version control.

## Features

### Module-Based Export
- Export Custom Fields and Property Setters by module
- Export Server Scripts and Client Scripts by module
- Export Custom HTML Blocks with module selection
- One-click export from form views

### Export Locations
- Custom Fields & Property Setters: `{app}/{module}/custom/{doctype}.json`
- Scripts & HTML Blocks: `{app}/fixtures/{doctype}.json`

### Bulk Export
- Bulk export multiple custom fields and property setters
- Export all fixtures for an app
- Grouped by module and DocType
